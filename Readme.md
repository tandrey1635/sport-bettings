# Спортивный портал «Профиль игрока» — статистика и достижения
Сайт внутренней системы для отображения статистики игрока, достижений, игровых показателей и истории сезонов на чистом HTML5/SASS с адаптивным дизайном.


## 🛠 Технологический стек

### Frontend
- **HTML5** — семантическая верстка, БЭМ-методология
- **SASS/SCSS** — сброс стилей, медиа-запросы
- **JavaScript** — работа с DOM, обработка событий


## 🎯 Ключевые функции

### UI/UX элементы
- Адаптивное бургер-меню для мобильных устройств
- Кастомные выпадающие списки (дропдауны)
- Карточка игрока с рейтингом и достижениями
- Таблица с показателями по сезонам
- Счетчики игр, голов, пасов и очков


## 🔧 Технические особенности

### Адаптивность
- Адаптивный дизайн
- Медиа-запросы
- Кросс-браузерная совместимость


**Проект демонстрирует навыки разработки сложных интерфейсов спортивных порталов с адаптивом, таблицами и кастомными дропдаунами.**
